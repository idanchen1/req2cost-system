# Req2Cost - RCWS Cost Estimation Platform

## Overview
Req2Cost is a full-stack web application designed to bridge the gap between operational requirements and engineering architecture for Remote Controlled Weapon Stations (RCWS). The platform allows users to select high-level operational needs (e.g., FIZ, FPZ, Auto Track), automatically maps them to specific hardware/software engineering blocks, and generates an accurate cost estimation.

## Key Features
* **Dynamic Cost Calculator:** Translates operational checkboxes into system components and calculates total costs on the fly.
* **Database Integration:** Powered by a PHP backend connected to a MySQL database to securely save, manage, and retrieve project histories.
* **Classified 'Secret Mode':** A hidden UI toggle feature that reveals restricted system capabilities (e.g., AESA Search Radar, Autonomous Fire) for authorized personnel only.
* **Responsive & Professional UI:** A clean interface tailored for defense and engineering contexts, built with modern web standards.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Database:** MySQL
